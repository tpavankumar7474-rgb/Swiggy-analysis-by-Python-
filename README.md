Food Delivery Sales Analysis Dashboard
Project Overview

This project analyzes a food delivery dataset using Python for data cleaning and analysis, and Tableau for creating an interactive dashboard. The objective is to identify sales trends, customer behavior, and business insights that can help improve decision-making.

Objectives
Analyze monthly and quarterly sales trends.
Compare revenue from Veg and Non-Veg food.
Identify top-performing states.
Analyze revenue by weekday.
Evaluate customer ratings.
Prepare a clean dataset for Tableau visualization.

Rating

Data Cleaning

The following preprocessing steps were performed:

Converted Order Date to datetime format.
Created YearMonth for monthly analysis.
Created Quarter for quarterly analysis.
Extracted weekday names.
Categorized dishes into Veg and Non-Veg using keyword matching.
Removed/handled missing values where required.
